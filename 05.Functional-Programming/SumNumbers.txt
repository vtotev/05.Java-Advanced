Sum Numbers
-----------------
Write a program that reads one line of Integers separated by ", ". Print the count of the numbers and their sum.
Use a Function<String, Integer>

Examples

Input
-----------------
4, 2, 1, 3, 5, 7, 1, 4, 2, 12

Output
-----------------
Count = 10
Sum = 41


Input
-----------------
2, 4, 6

Output
-----------------
Count = 3
Sum = 12
